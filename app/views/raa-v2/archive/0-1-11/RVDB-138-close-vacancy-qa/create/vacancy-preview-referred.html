  • What you will do in your working day

    As a Level 3 Software Developer Apprentice, you will be undertaking a recognised apprenticeship qualification, including Microsoft certifications, at the same time as working as part of the company’s Software Development team.

  • The training you will be getting

    You’ll work as part of the team to learn about the latest technologies used to deliver what is needed, including C# within the MVC framework, supported by the use of test driven development (TDD), continuous integration using Team City and continuous deployment using Octopus Deploy.

  • What to expect at the end of your apprenticeship

    As a Microsoft Software Developer Apprentice your main role and responsibilities will include:

    • Learning how to collate feature requests
    • Providing frontline helpdesk support
    • Growing into projects
    • Basic Programming
    • Becoming a valued part of the Software Development team
    • Helping existing staff with front-end development using Octopus Deploy Technology
    • Being responsible for extracting data and processing the extracted data
