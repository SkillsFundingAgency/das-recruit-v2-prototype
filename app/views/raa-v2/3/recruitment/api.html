Recruitment API

Integrating with our API may be useful if you are a large employer and want to create adverts using your existing content management system. It will allow you to publish to multiple sites, as well as Find an apprentice. 

You can also use the API to display all your adverts on any website.

What the API does

APIs let applications share information and data with each other. 

If you connect using our API, you can use your own content management system to create adverts on find an apprenticeship. 

You can also use our API to display your adverts on your own website. 

How to use the API

You'll need someone with technical knowledge to connect your software program with the API, for example a software developer.

You can share this technical documentation with the person who is going to connect your software program with the API. 

It can take a while to connect using an API, so leave enough time to integrate it. 

Only accounts with owner permissions can access the API. 

Accessing the API

If you have owner permissions you can access the API homepage. 

You'll need someone with technical knowledge to connect your software program with the API, for example a software developer.

Your developer will be able to try out the API using the sandbox provided.

Go to API homepage.

Get help and support

If you have any issues, either during development or once your software is live, you can get help and support from us.
